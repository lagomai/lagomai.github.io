ComfyUI安装 渠道

  • 官方 Github：https://github.com/comfyanonymous/ComfyUI
  • 秋叶整合包：https://pan.quark.cn/s/64b808baa960

具体安装步骤（官方）

此处只演示官方渠道，秋叶大佬的整合包可在上面地址前往下载，使用也比较简单，就不一一列举了，喜欢自己动手的可以跟着我的步骤来

打开官方网址：https://github.com/comfyanonymous/ComfyUI

往下滑，找到 Installing 部分，点击Direct link to download下载官方压缩包

解压后的文件内容如下，双击文件run_nvidia_gpu.bat即可启动Comfyui

启动速度非常快，会自动弹出浏览器操作页面

一个简单的解压，ComfyUI官方包就部署好了，它现在还是最原始的模样，接下来我们再做一些必要的操作，让我们使用起来更加方便。


安装 ComfyUI-Manager

打开Github 地址：https://github.com/ltdrdata/ComfyUI-Manager

点击”download ZIP”,下载到本地，之后将文件复制到【custom_nodes】目录，记得要解压，然后重新启动comfyui

无法安装解决办法：Git 未正确安装 | Git 下载

可以手动安装下GTI源代码管理系统https://git-scm.com/downloads/win，然后在文件资源管理器应用程序中，导航到 ComfyUl>custom nodes，在地址栏输入CDM，回车，在出现的命令提示符终端输入

完成后重启comfyui就有了，然后再用节点管理器安装下汉化就变成中文了


ComfyUI 界面汉化

打开Github 地址：https://github.com/AIGODLIKE/AIGODLIKE-ComfyUI-Translation

点击”download ZIP”,下载到本地，之后将文件复制到【custom_nodes】目录，记得要解压
